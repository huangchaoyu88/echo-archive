# 回聲資料室 / Echo Archive v2

原創繁體中文網頁 ARG。玩家瀏覽一個於 2008 年停止更新的虛構地方文化機構網站，從照片、名冊、廣播站紀錄、錄音帶、改稿備忘錄與還原清單，找回一名被從地方歷史中抹去的人。

> 本作品完全虛構。所有機構、人物、地點與事件皆為原創，不對應任何真實單位或案件。

## v2 主題

1993 年白礁里撤離名冊記載 46 名安置戶，但一張活動合照中出現 47 人。玩家將逐步確認第 47 人「周月琴」的身分、她保存的 A-17 錄音，以及 2008 年資料復原時發生的選擇性刪除。

最終目標不是揭露她現在的姓名與居所，而是把她在 1993 年的歷史位置恢復，同時尊重她要求保護現況與家人的意願。

## 公開版本

此儲存庫只包含玩家可取得的網站檔案與原創圖像資產，不包含完整攻略、故事聖經、答案地圖、真相時間線或內部 QA 文件。
