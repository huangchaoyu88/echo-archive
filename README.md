# 回聲資料室 / Echo Archive

原創繁體中文網頁 ARG 解謎遊戲。

玩家將調查一個於 2008 年停止更新的虛構機構網站，從公告、檔名、備份、報修紀錄、系統日誌與版本差異，重建一場數位典藏事故。

> 本作品完全虛構。所有機構、人物、地點與事件皆為原創，不對應任何真實單位或案件。

## 遊玩

GitHub Pages：https://huangchaoyu88.github.io/echo-archive/

- 遊玩時間：約 60–120 分鐘
- 語言：繁體中文
- 支援：桌面與手機瀏覽器
- 進度：儲存在瀏覽器 localStorage

## 公開版本

此儲存庫只包含玩家可取得的網站檔案，不包含完整攻略、故事聖經、答案版謎題文件或內部測試紀錄。
